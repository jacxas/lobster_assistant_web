# 🦞 Lobster Assistant

Asistente de IA local con integración multi-plataforma.

## Características

- 🤖 Motor de IA local (Ollama)
- 💬 Discord, Telegram, WhatsApp
- 🌐 Panel web de administración
- 🔒 100% privado, sin datos en la nube

## Instalación rápida

```bash
pnpm install
pnpm run setup
pnpm run dev
```

## Estructura del proyecto

```
lobster_assistant_web/
├── client/         # Frontend React + Vite
├── server/         # Backend API
├── shared/         # Tipos y utilidades compartidas
├── src/bot/        # Bots de mensajería
└── docs/           # Documentación
```
